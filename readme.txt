== Installation ==

1. Upload the plugin odk-call-action files to the `/wp-content/plugins/` directory
1. Activate the odk-call-action plugin through the 'Plugins' menu in WordPress.
1. Select menu ODK Call Action and insert phone number + popup for desktop
